## 2-3 良好的 PRD 寫法與自動接受模式

- PRD.md 檔內容：

    ```md
    # 產品描述

    請用Python寫一個可以在命令列執行的郵遞區號小幫手：

    - 使用 zip5.5432.tw 提供的 API 查詢台灣地址對應的郵遞區號。
    - 輸入是地址。
    - 輸出是 6 碼郵遞區號，特別留意 API 的傳回資料有 5 碼與 6 碼的郵遞區號，一定要確認取得其中 6 碼的郵遞區號。

    # 測試

    - 請以底下的地址確認可以得到 6 碼的郵遞區號：
        
        ```
        臺北市杭州南路一段 23 號
        ```
    ```
